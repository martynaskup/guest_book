# guest_book
A JavaScript based website to display guest book entries and let guests to leave their comments.


<h1>Features</h1>
<ul>
  <li>Add a new entry into the database</li>
  <li>Show all guest book entries</li>
</ul>

<h1>Technologies</h1>
<ul>
  <li>HTML5</li>
  <li>CSS3</li>
  <li>JavaScript</li>
  <li>Firebase</li>
  <li>date-fns library</li>
</ul>
